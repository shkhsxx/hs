## [케이스 01] 기능명: 로그인 후 대시보드 자동 진입

### 기능 정의
로그인 완료 시 대시보드 화면으로 즉시 이동하며, 미인증 사용자가 대시보드 URL에 직접 접근하면 로그인 페이지로 리다이렉트한다.
